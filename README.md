🎮 Gamer God
Uma aplicação web interativa que responde suas dúvidas sobre videogames utilizando inteligência artificial. Pergunte sobre estratégias, dicas, personagens e muito mais!

📋 Sobre o Projeto
Gamer God é uma aplicação desenvolvida em JavaScript vanilla que integra uma API de inteligência artificial para fornecer respostas detalhadas e úteis sobre qualquer jogo. Perfeita para gamers que buscam dicas de estratégia, walkthroughs ou informações gerais sobre seus títulos favoritos.
✨ Características Principais

🤖 Integração com IA: Respostas inteligentes e contextualizadas sobre jogos
🎨 Interface Modern Dark Mode: Design elegante e amigável ao usuário
⌨️ Input em Tempo Real: Digite suas perguntas e receba respostas instantâneas
📱 Responsivo: Funciona perfeitamente em desktop e dispositivos móveis
🚀 Performance Otimizada: Carregamento rápido e sem lags
🎯 UX/UI Limpa: Foco na usabilidade e experiência do usuário

🛠️ Tecnologias Utilizadas
Frontend

HTML5 - Estrutura semântica e moderna
CSS3 - Styling avançado com animações e responsividade
JavaScript (Vanilla) - Lógica de aplicação sem dependências desnecessárias

Backend & Integração

REST API - Integração com serviço de inteligência artificial
Fetch API - Requisições HTTP assíncronas

Ferramentas

Git & GitHub - Controle de versão
npm - Gerenciamento de dependências

🚀 Como Usar
Pré-requisitos

Navegador web moderno (Chrome, Firefox, Safari, Edge)
Conexão com internet (para requisições à API)

Instalação

Clone o repositório

bashgit clone https://github.com/Raulgonc/Gamer-God.git
cd Gamer-God

Instale as dependências (se houver)

bashnpm install

Abra no navegador

bash# Abra o arquivo index.html diretamente no navegador
# Ou use um servidor local
npx http-server
💻 Como Funciona

Digite sua pergunta no campo de input
Clique no botão "Perguntar" ou pressione Enter
Aguarde a resposta da inteligência artificial
Leia a resposta na área de conteúdo abaixo
Faça novas perguntas sem limite!

Exemplos de Perguntas
"Dicas Zelda: a link to the past"
"Qual é a melhor build para mago em Dark Souls?"
"Como desbloquear o final secreto em Elden Ring?"
"Estratégias para vencer o chefe final de Hollow Knight"

🔧 Configuração da API
Para usar a aplicação, você precisa configurar sua API key:

Obtenha uma API key em um serviço de IA (ex: OpenAI, Hugging Face, etc)
Adicione a chave no arquivo api/config.js:

javascriptconst API_KEY = 'sua_chave_aqui';

🐛 Troubleshooting
Problema: A API não responde

Verifique sua conexão com internet
Confirme que a API key está correta
Verifique o console do navegador (F12) para erros

Problema: Interface lenta

Limpe o cache do navegador
Desative extensões que possam interferir
Tente em outro navegador

Problema: Texto não aparece

Atualize a página
Verifique se os arquivos CSS estão carregados
Abra o DevTools para verificar erros
